Question-1: What is the different ways to select an element in the DOM?

Ans: There are many ways to select an element in the DOM.
For example:- 1. getElementById, 2. getElementByClassName, 3.querySelector, 4. querySelectorAll etc .

Question-2:What is the difference between innerHTML, innerText, textContent?

Ans:When you need to dynamically add or modify, then use innerHTML. innerText is used for visible text and textContent is used when needing all text, including hidden text .

Question-3:What is event delegation in the DOM?

Ans:Event Delegation is a technique where a single event listener is added to a parent element to handle events on multiple child elements, even if they are added dynamically.

Question-4:What is event bubbling in the DOM?

Ans:Event Bubbling is a phase in event propagation where an event starts from the target element and bubbles up to its parent elements.

Question-5:How do you create, add, and remove elements using JavaScript?
